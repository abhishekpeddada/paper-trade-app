# google-services.json Configuration

This file contains your Firebase Android configuration.

## How to get your google-services.json:

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Select your project
3. Click the gear icon â†’ **Project settings**
4. Scroll to "Your apps" section
5. Find your Android app (or add one with package name: `com.example.paper_trade_app`)
6. Click **Download google-services.json**
7. Copy it to this directory: `android/app/google-services.json`
